## 🤖 Project 1: The Rule-Based Guardrail

**Live Deployment:** https://yap-buddy-bot.lovable.app/

Project 1 focuses on building a deterministic AI guardrail rather than a conventional chatbot. Inspired by enterprise AI safety frameworks such as **NVIDIA NeMo Guardrails** and **Llama Guard**, the application acts as a security layer placed in front of a Large Language Model (LLM). I implemented a customer support assistant for **YAP Pakistan**, an SBP-licensed Electronic Money Institution (EMI), where every response is generated using predefined rules to ensure predictable, explainable, and compliant behavior.

### Key Learnings

- Rule-based systems provide deterministic and explainable responses.
- The chatbot follows a strict dictionary of predefined intents and responses.
- Any query outside the predefined rule set can be delegated to an LLM for natural language understanding.
- A guardrail acts as a safety layer before an LLM instead of replacing it.
- Guardrails should avoid web scraping or generating unverified information.
- They play a critical role in regulatory compliance, security, and enterprise AI governance.

## 📊 Project 2: Data Classification Using AI

Project 2 introduced the complete supervised machine learning pipeline through a classification problem. I learned how to prepare structured datasets, split data into training and testing sets, train a classification model, evaluate its performance, and make predictions on unseen data. This project helped me understand the difference between rule-based systems and data-driven AI models while establishing the fundamentals of supervised learning.

## 💼 Project 3: Tech Stack Recommender

Project 3 focuses on building a recommendation engine that maps a user's technical skills to suitable software engineering career paths. Using a structured dataset of job roles and required skills, I implemented a similarity-based matching engine that compares user skills against multiple roles, calculates matching scores, ranks recommendations, and highlights both matched and missing skills. This project strengthened my understanding of recommendation systems, similarity matching, and practical AI decision logic.

## 🖼️ Project 4: Image and Text Recognition

Project 4 demonstrates how pre-trained AI models can be integrated into a complete application without performing model training. I implemented both object detection and optical character recognition (OCR) by combining modern computer vision libraries with pre-trained deep learning models. The system detects objects within an image, extracts readable text, and presents both outputs through a unified workflow, providing practical experience with real-world AI model deployment and inference.
